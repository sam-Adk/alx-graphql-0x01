# Rick and Morty GraphQL App

## 🧠 Objective
Set up a Next.js application with TypeScript, Apollo Client, and Tailwind CSS to consume the Rick and Morty GraphQL API.

## ⚙️ Technologies Used
- Next.js
- TypeScript
- Apollo Client
- GraphQL
- Tailwind CSS
- ESLint
